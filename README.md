# 1️⃣ Sales Performance Dashboard – Power BI Project

## Project Overview
This dashboard analyzes 5,000+ sales records across UAE, Saudi Arabia, Qatar, and Oman regions. 
It tracks revenue, profit, product categories, and monthly trends for business insights.

## Tools Used
- Power BI
- DAX
- Power Query (ETL)
- SQL
- Excel

## Key Features
- KPI Cards: Total Revenue, Profit, Profit Margin %
- Monthly Revenue Trend (Line Chart)
- Revenue by Region (Bar Chart)
- Top Product Categories Table
- Dynamic Filters: Region, Date

## Business Impact
- Reduced manual reporting time by 40%
- Enabled data-driven decision-making
- Identified high-performing regions

## Screenshots
![Sales Dashboard](images/sales_dashboard.png)

## Files
- Dataset CSV: `PowerBI_Sales_Dataset.csv`
- Power BI File: `Sales_Dashboard.pbix`
